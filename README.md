# forja-dos-narradores
O Blog da Forja dos Narradores, um site para dicas, recomendações, e um toque de programação, com foco em RPG de mesa.

Em um repositório monorepo, é uma aplicação Full-Stack Client-Server. O frontend foi desenvolvido em Angular e o backend em NestJS, expondo uma API RESTful. O projeto funciona como um Blog/CMS onde usuários comuns podem ler e consumir postagens, enquanto administradores possuem um painel protegido para criação e gerenciamento do conteúdo e de categorias.
